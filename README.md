# fmif_programming_technologies_LR_2
Репозиторій для лабораторної роботи #2  (по докеру)



### Prerequirenments
Docker вже має стояти та бути налаштованим

### Запуск
1. `docker build -t docker-date-example . `

2. `docker run --rm -it docker-date-example`
